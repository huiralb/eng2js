Route

Fungsi route pada umumnya yaitu pengaturan event berdasarkan kriteria url yang terjadi perubahan.

Contoh Pembuatan Route : 
    
eng.route('/route2', function(param){
    return {
        param: param,
        controller: 'ctrls',
        target: 'route1',
        templateUrl: 'test-route2.php'
    };
});
eng.route('/*', function(param){
    return {
        param: param,
        controller: 'ctrls',
        target: 'route1',
        templateUrl: 'test-route.php'
    };
});
eng.controller('ctrls', function($root){
    this.name = 'ukung';
});
