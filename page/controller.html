Controller

Sebuah fungsi yang mengatur data html dengan tag "controller", ketika anda menuliskan tag html dengan attribute controller maka secara otomatis kompiler akan menerjemahkan hal apa saja yang ada di ruang lingkup data html controller tersebut.

Didalam satu dokument anda bisa membuat banyak controller sesuai dengan kebutuhan anda, dan setiap controller bisa saling berkomunikasi dengan menggunakan banyak cara, bisa dengan config, service, module atau factory atau ada juga beberapa model object yang bisa di include didalamnya dengan tehnik yang anda akan buat nantinya.
