Dokumentasi Eng2Js

The Magic Quote ˆ

Dokumentasi singkat ini dibuat khusus untuk pemograman web dengan menggunakan javascript sebagai backend-nya. Eng2Js merupakan Javascript biasa yang di build menjadi sebuah kerangka kerja untuk membantu developer membuat applikasi. disertai dengan contoh-contoh yang sifatnya mendasar dan terstruktur, terinspirasi dari beberapa framework yang sudah ada dan di kembangkan sesuai konsep yang mudah di fahami.

Eng2Js membutuhkan Library terbaik yaitu jQuery, anda bisa menggunakan jQuery versi berapapun

Sejarah singkat Eng2Js
Eng2Js versi pertama dengan kode Versi 0.0.1 di buat pada tahun 2015 hadir untuk memenuhi kebutuhan pemograman web dan fokus pada penggunaan HTML seutuhnya dan javascript sebagai pelengkap sebuah konsep yang sudah cukup familiar di beberapa framework js sebelumnya di bangun ulang dan dikonsep ulang dengan konsep yang sangat berbeda lebih dalamnya tujuanya untuk mempermudah penggunaan dan penerapan di dalam pembuatan Applikasi yang sifatnya komplex. disamping itu sudah mendukung konsep MVC didalamnya.
