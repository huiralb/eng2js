Sejarah singkat Eng2Js
Eng2Js versi pertama dengan kode Versi 0.0.1 di buat pada tahun 2015 hadir untuk memenuhi kebutuhan pemograman web dan fokus pada penggunaan HTML seutuhnya dan javascript sebagai pelengkap sebuah konsep yang sudah cukup familiar di beberapa framework js sebelumnya di bangun ulang dan dikonsep ulang dengan konsep yang sangat berbeda lebih dalamnya tujuanya untuk mempermudah penggunaan dan penerapan di dalam pembuatan Applikasi yang sifatnya komplex. disamping itu sudah mendukung konsep MVC didalamnya.

Penggunaan Single Page Application semakin populer baik di ruang lingkup website ataupun di ruang lingkup applikasi dekstop, pemilihan alat yang tepat sangat dianjurkan untuk memenuhi kebutuhan applikasi kedepanya pembangunan framework ini diharapkan akan terus berlanjut untuk semakin melengkapi karena sudah mendukung module terpisah sehingga mempermudah developer saling membagi kode program dengan lainya, membangun sebuah komunitas yang didalamnya bisa saling membantu.
