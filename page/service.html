Service

Service konsepnya hampir sama dengan config, hanya saja service adalah sebuah Funtion sehingga untuk applikasi yang sifatnya membutuhkan konfigurasi awal metode ini sangat membantu. semua property yang di miliki service bisa di akses dan dirubah oleh setiap controller.

Berikut adalah contoh service: 

    
eng.service('$srv', function(){
    var dataCounter = {
        count: 0,
        getCounters: function(){
            return this.count++;
        }
    };
    dataCounter.getCounters();
    return dataCounter;
});
eng.controller('ctrl-1', function($root, $srv){
    $root.pushCounter = function(){
        console.log( $srv.getCounters() );
    };
});
eng.controller('ctrl-2', function($root, $srv){
    $root.pushCounter = function(){
        console.log( $srv.getCounters() );
    };
});
