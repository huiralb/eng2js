Tag Directive

Tag Directive hampir sama dengan directive hanya saja bermain di segment attribute html. alasan kenapa di pisahkan dan tidak menggunakan "restrict" adalah untuk mempermudah dan memisahkan kedua fungsi tersebut sehingga tidak membingungkan.

Contoh Pembuatan Tag Directive : 
    
eng.tagDirective('tampil', function(value, root, element){
    if(value && typeof value === 'boolean'){
        $(element).show();
    }else{
        $(element).hide();
    }
});
// ketikan di browser
