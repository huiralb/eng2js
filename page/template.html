Template

Di dalam Eng2Js v 0.0.1 template view diambil dari external code dengan metode Ajax

Contoh penggunaan view



Ada method success, start dan error jika view selesai mengambil kontentnya. 

eng.controller('sample123', function($root){
    $root.successLoadView = function(){
        alert( "Oke..." );
    };
    $root.startLoadView = function(){
        alert( "Start..." );
    };
    $root.errorLoadView = function(){
        alert( "Error..." );
    };
});

Beberapa events yang ada di view
  • success
  • start
  • error

Settingan cache content di bisa di setting di tag-nya dengan settingan sebagai berikut


Sedangkan settingan cache local agar ajax tidak direload kembali adalah dengan memberikan tag local-cache="true",
