Event

Event pada Eng2Js didalam controller ditandai dengan tidak dipakainya prefix "on" misalnya pada event "onclick" menjadi "click".
kemudian object this menjadi $$this dan event/evt yang secara default terdapat pada argument event digantikan dengan $$event.
atau langsung memberikan nilai didalam fungsi event tersebut, anda bisa melihat banyak sekali digunakan didalam contoh di folder sample disini
