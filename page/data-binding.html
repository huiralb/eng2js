Data Binding

Didalam penulisan data model binding ada 2 macam yaitu anda bisa langsung menggunakan TAG html atau didalam controller. berikut contoh jika anda melakukanya didalam tag html

Lihat contoh

init pada init="nama='jhon'" akan di olah oleh controller sebagai $root.nama = 'jhon'; masukan data init bisa berupa String Numbers Array Boolean atau Object

Model nama didalam input akan langsung diolah dan di implementasikan ke tags yang berkaitan
