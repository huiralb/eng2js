Tentang Eng2Js

Pengembang terinspirasi oleh beberapa framework engine javascript yang sudah ada sebelumnya di kombinasikan dengan syntax dan konsep yang lebih sederhana dan mudah di pahami untuk programmer namun tetap memikirkan kestabilan data-binding yang menjadi fokus utama membangun framework ini.

Framework ini dibangun dari dasar dengan javascript object yang dimanupulasi sedemikian rupa dan diolah didalam controll data yang saling berkaitan. pendekatan kepada HTML5 men sumber dayakan tag dan memodifikasi penulisan HTML menjadi lebih interaktif.

Dalam penggunaan frameworknya pun terbilang sangat mudah setelah melalui beberapa uji coba terhadap orang yang belum mengerti pemograman web.

Harapan framework ini dibuat adalah untuk membuat banyak pilihan kepada programmer web khususnya untuk membuat sebuah applikasi yang komplex dan semua dokumentasi ditulis dengan bahasa lokal walaupun istilah didalam penulisan kodenya tidak menggunakan bahasa lokal tapi menggunakan bahasa yang umum digunakan gunanya untuk pendekatan ke aray systematis.
