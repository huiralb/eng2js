Filter

Filter adalah metode pemerosesan output kembali yang telah di hasilkan oleh system data output bisa berupa object atau string number dan boolean.

filter juga bisa di kembangkan lagi atau dibuat filter yang mungkin anda butuhkan yang secara default tidak ada di dalam bawaan framework.

Contoh Pembuatan Filter : 
    
eng.filter('balik', function(text, config){
    if(config.upper){
        return text.split('').reverse().join('').toUpperCase();
    }else{
        return text.split('').reverse().join('');
    }
});

// ketik di dokument controller
// ˆ'Hello'|balik
